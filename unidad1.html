1.2 Organización de los datos en la memoria física y lógica

La memoria (RAM) esta físicamente formada por bits, componentes que pueden estar en uno de dos estados claramente definidos, encendido/apagado 0’s y 1’s. Como se observa un bit puede representar dos valores, combinando grupos de bits se pueden representar más estados. La memoria física RAM está compuesta por casillas de memoria en las cuales se pueden almacenar datos que están organizados por direcciones de memoria. Cada celda en una dirección puede ser de 8, 16, 32 o 64 bits.

Ejercicio

Realizar el siguiente ejercicio:

  1. Para almacenar el sexo de una persona ¿cuántos bits se requieren?
  2. Cuántos bits se requieren para almacenar los siete días de la semana?
  3. Cuántos bits se requieren para almacenar los meses del año?
  4. Para almacenar el código ASCII ¿cuántos bits se requieren?

1.2.1 Clasificación de datos simples y compuestos

De acuerdo con la forma en que los datos se organizan, se clasifican en:

a) Tipos de datos simples
b) Tipos de datos compuestos o estructurados


a) Un tipo de dato simple (primitivo) es aquel que hace referencia a un único valor a la vez y que ocupan solo una casilla de memoria, por tanto, una variable simple hace referencia a un único valor a la vez.

En este grupo se encuentran:

  • Enteros (int): números sin decimales, como 10, -3 o 2025
  • Flotantes (float o double): números con punto decimal, como 3.14 o -0.25
  • Caracter (char): un solo símbolo, como ‘a’, ‘9’ o ‘$’
  • Booleanos (bool): representan valores lógicos: verdadero (true) o falso (false).

Direccionamiento de la memoria
La unidad de control se dirige a los distintos bytes de memoria por su posición. Todos los lenguajes de programación permiten dar un nombre a las posiciones de memoria y se conocen como variables. Entonces una variable son las posiciones de memorias asociadas a un nombre, por ejemplo: Area, Base y Altura. Las variables pueden cambiar su contenido o pueden contener un valor constante, como es el caso del valor pi.

Usando un lenguaje de programación, por ejemplo, c++, se declaran las variables asignando un valor a las variables Base y Altura.

      
        int Base = 3;
        int Altura = 2; 
        int Area;

Automáticamente la unidad de control asigna a las casillas de memoria con sus direcciones a esas variables. Al momento de declarar las variables de tipo de dato entero esas casillas nada más van a permitir datos enteros, en el caso del ejemplo al momento de declararlas se les ASIGNÓ un valor de tipo de dato entero a las variables Base=3 y Altura=2.

1.3 Representación de las estructuras de datos

Las estructuras de datos son fundamentales para las principales actividades para llegar al desarrollo de grandes sistemas de software. Las estructuras de datos permiten organizar, almacenar y manipular datos en la memoria de una computadora.


1.3.1 Operaciones básicas sobre las estructuras de datos

Las operaciones que se pueden hacer con estructuras de datos son varias. Las más básicas son:

  • Insertar : Agregar un valor a la estructura.
  • Eliminar : Borrar un valor existente.
  • Buscar : Encontrar un valor específico.
  • Actualizar: Modificar un valor existente.
  • Recorrer : Acceder a cada elemento de la estructura.
